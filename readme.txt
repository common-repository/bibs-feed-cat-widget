Places a Widget with categories and feeds for each category in your sidebar

== Description ==

A very simple widget which shows all your categories with number of posts togehther with RSS Feeds for each category in your WordPress sidebar.

Find more infos about Bibs Feed Cat Widget at <a href="http://wp-ezine.com/bibs-feed-cat-widget_25300.html">WP Ezine</a>

== Installation ==

Install

1. Upload cat-feeds.php to the /wp-content/plugins/bibs-cat-feed directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the Widget in your sidebar . Done

== Frequently Asked Questions ==

Question: Can i exclude categories? 

Answer: Yes, just modify the patameters in the wp_list_categories tag in the plugin code.
